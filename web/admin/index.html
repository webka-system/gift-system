商品の登録

QRコード一括生成

種別を指定して任意個数のQRカードを生成します。トークンはサーバ側で推測不可能な値として発行されます。

印刷用URL一覧（Excel）

対象のQRカードの受け取り者URL（/g/<token>）を1行ずつ並べた Excel を出力します。印刷工場はこのURL列を使ってQR化・印刷を行います。既定は A列=URL / B列=token / C列=種別名。
